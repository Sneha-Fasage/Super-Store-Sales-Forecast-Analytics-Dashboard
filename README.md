# Super-Store-Sales & Forecast-Analytics-Dashboard
1. Project Overview

Super Store Sales Analytics with 15-Day Forecasting.
An end-to-end business intelligence solution developed in Power BI to analyze historical sales data, uncover key business insights, and generate short-term sales forecasts to support strategic decision-making.

3. Business Problem

Organizations often struggle to track performance across multiple dimensions such as region, category, and customer segments while also lacking predictive insights for future planning.

This project addresses:-

Lack of centralized sales performance monitoring.
Limited visibility into profit-driving segments.
Inability to forecast short-term demand accurately.

3. Solution

Developed an interactive and scalable Power BI dashboard that:-
Tracks key KPIs in real-time.
Provides multi-dimensional analysis (time, geography, category).
Enables drill-down insights for granular decision-making.
Implements time-series forecasting for future sales prediction.

4. Tech Stack

Power BI Desktop – Dashboard development & data visualization.
Power Query (ETL Layer) – Data cleaning, transformation, and modeling.
DAX (Data Analysis Expressions) – Advanced calculations, KPIs, and measures.
Microsoft Excel / CSV – Data source.
Data Modeling – Star schema approach for optimized performance.
File Formats – .pbix (development), .png (report snapshots).

5. Data Model & Preparation

Cleaned and transformed raw transactional data using Power Query.
Handled missing values and ensured data consistency.
Created relationships between fact and dimension tables.
Built calculated columns and measures using DAX.

Key Measures Created:-
Total Sales.
Total Profit.
Profit Margin.
Average Delivery Time.
Sales Growth %.
Forecasted Sales.

7. Key Business Insights

Sales exhibit strong upward trends in Q4, indicating seasonal demand.
Consumer segment drives maximum revenue, highlighting B2C dominance.
Technology category contributes highest sales, making it a key revenue driver.
High reliance on Cash on Delivery (COD) suggests trust or accessibility factors.
Certain regions/states consistently outperform others, indicating market concentration.
Forecast suggests continued growth trend, supporting expansion strategies.

8. Business Impact
